# UNP MAPS

### Características

- inicio de sesión.
- Visualización de las instalaciones de la universidad politécnica de Nicaragua.
- Información detallada de los lugares.
- Reseñas de los usuarios.

###### Desarrollado con flask, sqlite, javascript
